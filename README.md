# slot-machine
Simple slot machine with famous quotes written in react.

## Versions
```
node: v12.22.9
npm: 8.5.1
```


## Build
```
npm run build
```

## Run the project
### Local
``
npm start
``
### Production
Use `/build/` as root directory in your virtual host configuration.
